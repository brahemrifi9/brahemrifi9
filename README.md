## 🔥 Featured Project — SalonFlow

![Status](https://img.shields.io/badge/Status-Live%20in%20Production-brightgreen?style=flat-square)
![Clients](https://img.shields.io/badge/Clients-1%20Live%2C%201%20Onboarding-blue?style=flat-square)

A **multi-tenant SaaS platform** for barbershops — built and deployed while in my second year of university. Each business runs on shared infrastructure with fully isolated data, its own WhatsApp booking flow, and a JWT-authenticated admin panel.

- 💬 WhatsApp chatbot booking flow (per-tenant, multilingual ES/EN)
- 🗓️ Real-time availability engine with overlap protection at DB level
- 🏢 Multi-tenant architecture — business isolation via `business_id` FK across all tables
- 🔐 JWT-authenticated admin dashboard per business
- ☁️ Deployed on Hetzner VPS with Docker, Nginx, and Cloudflare
- ⚙️ Database migrations managed with Alembic
- 🔒 Security hardened — UFW, Fail2Ban, HTTPS via Let's Encrypt
